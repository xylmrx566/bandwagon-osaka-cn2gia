## 十一、常见问题 FAQ

**Q1：JPOS_6 和 JPOS_1 是同一个机房吗？**
不是。JPOS_6 是 CN2 GIA 线路，JPOS_1 是软银 BBTEC 线路，物理位置都在大阪但走不同的网络。CN2 GIA 对电信用户最优，软银对联通用户更友好，价格也差不少（CN2 GIA 起步 $499.99/年，软银走 CN2 GIA-E 套餐起步 $169.99/年）。

**Q3：CN2 GIA 和 CN2 GT 有什么区别？**
CN2 GIA（AS4809 Global Internet Access）是电信最贵最稳的精品线路，CN2 GT（AS4809 Global Transit）是中间档，从 2019 年开始已经和普通 163 网差不多拥挤了。买搬瓦工认准 CN2 GIA 或 CN2 GIA-E，别买成 CN2 GT。

**Q4：30 天退款政策是真的吗？**

**Q5：优惠码失效了怎么办？**

**Q6：Osaka CN2 GIA 适合做网站 CDN 回源吗？**
适合，尤其是面向国内用户的网站。低延迟加低丢包，回源体验会比海外机房好不少。但流量配额要算清楚，CDN 回源是大流量场景，最低档 0.5 TB/月很容易跑爆。
